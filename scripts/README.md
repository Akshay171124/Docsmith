# Scripts

Operational and developer scripts, grouped by purpose (Forge convention).

- `data/` — mine git history for code+doc commit pairs used by the evaluation harness.
- `dev/` — local helpers (build the index for a repo, run the pipeline against a local
  diff, smoke-test a language's tree-sitter extraction).

These are tools around Docsmith, not part of the shipped package in `src/`.
